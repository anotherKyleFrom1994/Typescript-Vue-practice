# Typescript-Vue-practice

Below is what I use to construct this project:

## Framework & Language
  * Vue 2.x
  * Typescript 3.5x

## State managent
  * Vuex

## UI design
  * Vuetify 2.x

## Linting
  * ESLint

## Testing
  * jest(unit)
  * Nightwatch(e2e)
